### 2. 后端修复验证

| 缺陷ID | 模块 | 验证内容 | 预期结果 | 实际结果 | 状态 |
|--------|------|---------|---------|---------|------|
| BUG-003 | 知识库 | getKnowledgePointDetail() | 返回KnowledgePointVO | 返回null | ❌ 未修复 |
| BUG-005 | 学习计划 | /today-tasks接口 | 存在 | 不存在 | ❌ 未修复 |
| BUG-006 | 学习计划 | 今日任务专用接口 | 存在 | 不存在 | ❌ 未修复 |
| BUG-011 | 试卷 | submitExam()返回值 | ExamResultVO | Void(null) | ❌ 未修复 |

**验证详情**:

- **BUG-003**: `KnowledgeController.java:58-64`
  ```java
  @GetMapping("/points/{id}")
  public Result<KnowledgePointVO> getKnowledgePointDetail(@PathVariable Long id) {
      knowledgeService.incrementViewCount(id);
      return Result.success(null);  // ❌ 仍然返回null
  }
  ```
  缺陷依然存在，需要调用 `knowledgeService.getKnowledgePointDetail(id)` 并返回实际数据。

- **BUG-005 & BUG-006**: `PlanController.java`
  检查结果: 该Controller中不存在 `/today-tasks` 端点。只有以下端点:
  - `POST /` - 创建计划
  - `GET /` - 获取计划列表
  - `GET /{id}` - 获取计划详情
  - `PUT /{id}` - 更新计划
  - `DELETE /{id}` - 删除计划
  - `PUT /{id}/status` - 更新状态
  - `GET /{id}/tasks` - 获取指定计划的任务(需要planId)
  - `POST /tasks/{taskId}/execute` - 执行任务
  
  缺陷依然存在，需要添加 `GET /today-tasks` 接口。

- **BUG-011**: `PaperController.java:104-112`
  ```java
  @PostMapping("/exam/{examRecordId}/submit")
  public Result<Void> submitExam(...) {
      paperService.submitExam(...);
      return Result.success("提交成功", null);  // ❌ 返回null
  }
  ```
  缺陷依然存在，需要修改返回类型为 `Result<ExamResultVO>` 并返回得分信息。
