## 缺陷详情

### BUG-001: 前端章节接口路径错误

**缺陷ID**: BUG-001  
**模块**: 知识库  
**功能**: 获取章节树  
**严重程度**: 高  

**缺陷描述**:
前端API调用路径与后端Controller路径不匹配，导致前端无法正确获取章节数据。

**代码位置**:
- 前端: `frontend/src/api/knowledge.ts:10-11`
- 后端: `backend/src/main/java/com/studyhelper/controller/KnowledgeController.java:35-39`

**前端代码**:
```typescript
// 错误：使用了错误的路径
export const getChapters = (subjectId: number) => {
  return get<Chapter[]>(`/knowledge/subjects/${subjectId}/chapters`)
}
```

**后端实现**:
```java
// 正确的路径
@GetMapping("/chapters")
public Result<List<ChapterTreeVO>> getChapters(@RequestParam Long subjectId,
                                               @RequestParam(required = false) Integer grade) {
    List<ChapterTreeVO> chapters = knowledgeService.getChapterTree(subjectId, grade);
    return Result.success(chapters);
}
```

**预期结果**: 
前端调用 `/knowledge/chapters?subjectId=1`

**实际结果**: 
前端调用 `/knowledge/subjects/1/chapters`（错误路径）

**修复建议**:
```typescript
export const getChapters = (subjectId: number, grade?: number) => {
  return get<Chapter[]>('/knowledge/chapters', { subjectId, grade })
}
```

### BUG-003: 知识点详情接口未实现

**缺陷ID**: BUG-003  
**模块**: 知识库  
**功能**: 知识点详情  
**严重程度**: 严重  

**缺陷描述**:
知识点详情接口虽然增加了浏览次数，但实际返回的数据为null，核心功能未实现。

**代码位置**:
`backend/src/main/java/com/studyhelper/controller/KnowledgeController.java:58-64`

**问题代码**:
```java
@GetMapping("/points/{id}")
public Result<KnowledgePointVO> getKnowledgePointDetail(@PathVariable Long id) {
    // 增加浏览次数
    knowledgeService.incrementViewCount(id);
    // 实际应该返回详情，这里简化处理
    return Result.success(null);  // ❌ 返回null
}
```

**预期结果**:
返回完整的知识点详情，包含：
- title: 知识点标题
- content: 知识点内容
- difficultyLevel: 难度等级
- viewCount: 浏览次数
- relatedPoints: 相关知识点

**实际结果**:
返回 `{"code":200,"message":"success","data":null}`

**修复建议**:
```java
@GetMapping("/points/{id}")
public Result<KnowledgePointVO> getKnowledgePointDetail(@PathVariable Long id) {
    // 增加浏览次数
    knowledgeService.incrementViewCount(id);
    // 获取知识点详情
    KnowledgePointVO detail = knowledgeService.getKnowledgePointDetail(id);
    if (detail == null) {
        return Result.error(4001, "知识点不存在");
    }
    return Result.success(detail);
}
```

### BUG-005: 今日任务接口路径完全不一致

**缺陷ID**: BUG-005  
**模块**: 学习计划  
**功能**: 获取今日任务  
**严重程度**: 高  

**缺陷描述**:
文档定义的今日任务接口与实际实现完全不同，缺少核心功能。

| 项目 | 文档定义 | 实际实现 |
|-----|---------|---------|
| 路径 | GET /api/plan/today | GET /plan/{id}/tasks |
| 功能 | 获取所有今日任务 | 获取指定计划的某天任务 |

**文档定义** (`接口设计.md 6.5节`):
```
GET /api/plan/today-tasks
返回：taskId, planId, planName, subjectName, chapterName等完整任务信息
```

**实际实现**:
```java
@GetMapping("/{id}/tasks")
public Result<List<DailyTask>> getDailyTasks(HttpServletRequest request, @PathVariable Long id,
                                             @RequestParam(required = false) 
                                             @DateTimeFormat(pattern = "yyyy-MM-dd") LocalDate date)
```

**问题分析**:
1. 接口路径完全不同
2. 缺少专门的今日任务查询接口
3. 需要传递planId参数才能获取今日任务

**修复建议**:
添加专门的今日任务接口：
```java
@GetMapping("/today-tasks")
public Result<List<DailyTaskVO>> getTodayTasks(HttpServletRequest request) {
    Long userId = (Long) request.getAttribute("userId");
    List<DailyTaskVO> todayTasks = planService.getTodayTasks(userId);
    return Result.success(todayTasks);
}
```

| 项目 | 文档定义 | 实际实现 |
|-----|---------|---------|
| 路径 | POST /api/plan/complete | POST /plan/tasks/{taskId}/execute |

**文档定义**:
```json
POST /api/plan/complete
```

```java
@PostMapping("/tasks/{taskId}/execute")
public Result<Void> executeTask(HttpServletRequest request, @PathVariable Long taskId,
                                @RequestBody Map<String, Integer> params) {
    Long userId = (Long) request.getAttribute("userId");
    Integer actualTime = params.get("actualTime");
    Integer completedCount = params.get("completedCount");
    planService.executeTask(taskId, userId, actualTime, completedCount);
    return Result.success("任务执行成功", null);
}
```

**修复建议**:
统一接口设计，建议：
1. 保持 `/plan/tasks/{taskId}/execute` 路径（语义更清晰）
2. 更新文档以匹配实际实现

### BUG-011: 提交试卷结果未返回得分

**缺陷ID**: BUG-011  
**模块**: 试卷  
**功能**: 提交试卷  
**严重程度**: 严重  

**缺陷描述**:
试卷提交接口返回类型为Void，未返回得分和正确答案等关键信息。

**代码位置**:
`backend/src/main/java/com/studyhelper/controller/PaperController.java:104-112`

**问题代码**:
```java
@PostMapping("/exam/{examRecordId}/submit")
public Result<Void> submitExam(HttpServletRequest request, @PathVariable Long examRecordId,
                               @RequestBody Map<String, Object> params) {
    Long userId = (Long) request.getAttribute("userId");
    String answersJson = (String) params.get("answersJson");
    Integer duration = params.get("duration") != null ? ((Number) params.get("duration")).intValue() : 0;
    paperService.submitExam(userId, examRecordId, answersJson, duration);
    return Result.success("提交成功", null);  // ❌ 返回null
}
```

**文档定义响应**:
```json
{
  "code": 200,
  "message": "提交成功",
  "data": {
    "recordId": 4001,
    "totalScore": 100,
    "obtainedScore": 85,
    "correctCount": 17,
    "wrongCount": 3,
    "wrongQuestionIds": [2001, 2005, 2010]
  }
}
```

**修复建议**:
```java
@PostMapping("/exam/{examRecordId}/submit")
public Result<ExamResultVO> submitExam(HttpServletRequest request, @PathVariable Long examRecordId,
                                       @RequestBody Map<String, Object> params) {
    Long userId = (Long) request.getAttribute("userId");
    String answersJson = (String) params.get("answersJson");
    Integer duration = params.get("duration") != null ? ((Number) params.get("duration")).intValue() : 0;
    ExamResultVO result = paperService.submitExam(userId, examRecordId, answersJson, duration);
    return Result.success("提交成功", result);
}
```
